## Usage in Volum

For the React/Vite app, import SVG icons like:

```ts
import folderIcon from '../assets/places/22/folder.svg';
// or as a URL for <img>:
<img src={new URL('../assets/mimetypes/22/text-html.svg', import.meta.url).href} />
```

For inline SVG (allows CSS color control), use Vite's `?react` plugin or `?raw` inline option. Since these are KDE monochrome icons, inlining them as raw SVG strings and inserting into JSX is the most flexible approach.

Recommended sizes:
- **Toolbar buttons:** `actions/22`
- **File type icons in list:** `mimetypes/22`
- **Folder icons in sidebar:** `places/22`
- **File type icons in grid:** `mimetypes/32`
- **Status/notification:** `status/22`
- **Preview placeholder:** `mimetypes/64`
